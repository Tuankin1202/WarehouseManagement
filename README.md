# Warehouse Management System

Hệ thống quản lý kho hàng được xây dựng bằng Python và PostgreSQL.

## Chức năng

- Đăng nhập
- Xem sản phẩm
- Thêm sản phẩm
- Sửa sản phẩm
- Xóa sản phẩm
- Tìm kiếm sản phẩm
- Nhập kho
- Xuất kho
- Thống kê kho

## Công nghệ sử dụng

- Python 3
- PostgreSQL
- psycopg2
- Git & GitHub

## Cách chạy

Cài thư viện:

```bash
pip install psycopg2-binary
```

Chạy chương trình:

```bash
python main.py
```
